Task Tracker App

A simple Task Tracker App built using HTML, CSS, and JavaScript.  
It allows users to add, edit, delete, and search tasks with LocalStorage support.  

🚀 Features
- Add new tasks
- Mark tasks as complete
- Edit & Delete tasks
- Search tasks
- LocalStorage persistence
